# find_water
Модуль для обнаружения и сегментации водных объектов (рек, озер, водоемов) на спутниковых снимках с использованием нейросетевой архитектуры U-Net.

## Основные функции

  * Обучение модели с нуля на датасете S1S2-Water
  * Дообучение сохраненной модели на новых данных
  * Визуализация результатов сегментации
  * Боевое тестирование на пользовательских изображениях

## Структура модуля

### 1. Архитектура модели (U-Net)
```
class UNet(nn.Module):
    """
    U-Net для сегментации водных объектов
    - Вход: RGB изображение (3 канала)
    - Выход: бинарная маска воды (1 канал)
    - Размер: 256x256 пикселей
    """
```
### 
2. Режимы работы
|------|------|--------|
Режим	| Описание |	Использование|
Новое обучение	| Обучение модели с нуля	| Часть 1 датасета S1S2-Water | 
Дообучение	| Продолжение обучения на новых данных	| Загрузка модели + новая часть |
